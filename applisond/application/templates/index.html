Bienvenu sur l'application d'analyse de sondage

Ensemble de microservices réalisé par les étudiants dans le cadre d'un projet académique qui consiste à visualiser les résultats d'une enquete

L'objectif de ce projet est de développer une interface ou un outil de visualisation numérique présentant/affichant sur écran le résultat d'une enquête approfondie sur les habitudes de consommation/d'utilisation des utilisateurs d'applications mobiles sur la 4G et la 5G.

Utilisation de l'application

Une fois l'enquete, les réponses doivent etre récupéré dans un fichier csv, ensuite les l'utilisateur se connecte après avoir crée un compte, place le fichier csv dans le formulaire, en précisant la variable des données du sondage à visualiser. L'application affiche les graphes des données.

Consignes

Pour respecter l'alogorithme mis en place pour traiter les données de l'application, vos données doivent etre classées d'une manière particulière. Un modèle de fichier csv est téléchargeable comme exemple.
